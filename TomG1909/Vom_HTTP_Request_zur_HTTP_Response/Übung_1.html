HTTP Requests

  • GET-Request: Ist erfolgreich, Statuscode: 200
  • POST-Request: Statuscode: 404 - not found. Angaben im Header vorhanden
  • PUT-Request: Statuscode: 405 - Method not allowed und anschließend 404 - not found
  • DELETE-Request: Statuscode: 405 - Method not allowed und anschließend 404 - not found

Der GET-Request ist für das Abrufen von Informationen vom Server geeignet Der POST-Request ist am besten für das schicken neuer Daten, wie z.B. Angsben aus einem Formular, an den Server geeignet. Der PUT-Request ist am besten dazu geeignet bestehende Daten zu aktualisieren. Der DELETE-Request wird benötigt um Daten vom Server zu löschen.
